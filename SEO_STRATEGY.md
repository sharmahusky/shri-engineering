## SEO Objectives

1. Rank for industrial filtration keywords
2. Establish authority in equipment manufacturing
3. Drive qualified leads through organic search
4. Improve local search visibility

## On-Page SEO

### Meta Tags
- **Title**: 50-60 characters, include primary keyword
- **Description**: 150-160 characters, compelling CTA
- **Keywords**: 5-7 relevant keywords

### Content Structure
- H1: One per page, includes primary keyword
- H2/H3: Logical hierarchy, includes secondary keywords
- Internal Links: 3-5 per page to relevant content
- Image Alt Text: Descriptive, includes keywords

### Technical SEO
- Mobile-responsive design
- Fast page load (< 3 seconds)
- XML sitemap
- Robots.txt
- Structured data (JSON-LD)
- Open Graph tags
- Twitter Card tags
